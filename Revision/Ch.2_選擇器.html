Ch.2 css選擇器： A. 常用選擇器 1. 元素選擇器： - 通過標籤名來選中指定元素 - 語法： 元素標籤{} 2. id選擇器： - 通過id名來選中指定元素 - 語法： #id{} 3. class選擇器： - 它是一個標籤的屬性，與 id 類似，不同的是 class 可以重複使用 - 根據元素中的 class 屬性值選中一組元素 - 語法：.class{} 4. 同配選擇器 - 選擇頁面中所有元素 - 語法：*{} B. 選擇器語法延伸使用 1. 交集選擇器： - 選中同時複合多個條件的元素 - 語法：選擇器1選擇器2選擇器n{} 注意：以元素選擇器開頭 2. 並集選擇器： - 同時選擇多個選擇器對應的元素 - 語法：選擇器1, 選擇器2, 選擇器n{} C. 關係元素與關係選擇器： I. 關係元素 1. 父元素 包含子元素的元素 2. 子元素 被父元素包含的元素 3. 祖先元素 直接或間接包含後代的元素 4. 後代元素 直接或間接被祖先元素包含的元素 5. 兄弟元素 擁有相同父元素的元素 II. 關係選擇器 1. 子元素選擇器： - 指定父元素裏的子元素 - 語法：父元素>子元素 {} 注意，必須是子元素，不可是後代元素 2. 後代元素選擇器： - 選中指定元素內的所有指定後代元素（範圍更廣） - 語法：祖先元素 後代元素 {} 3. 兄弟元素選擇器： - 語法：前一個 + 下一個（一定要是下一個，緊接著的） - 語法：前一個 ~ 下面的所有 D. 屬性選擇器 - for

- 語法：元素標籤[title]{} 1. [屬性名] 選擇含有指定屬性的元素 2. [屬性名=屬性值] 選擇含有指定屬性和屬性值的元素 3. [屬性名^=屬性值] 選擇屬性值中以指定值開頭的元素 4. [屬性名$=屬性值] 選擇屬性值中以指定值結尾的元素 5. [屬性名*=屬性值] 選擇屬性值中含有指定值的元素 E. 偽類選擇器 - 用來描述一個元素的特徵/狀態 (如第一個子元素、被點擊的元素、鼠標移動的元素) - 語法：元素:xxx{} 1. :first-child 第一個子元素 2. :last-child 最後的子元素 3. :nth-child() - (x)第 x 個子元素 - 如果寫(n)則 0-無窮 個子元素 - (odd)/(2n+1) odd numbers - (even)/(2N) even numbers - 超連結的偽類選擇器 1. :link 表示未訪問過的連接 2. :visited 表示已訪問過的連接 3. :hover 表示鼠標移入 4. :active 表示鼠標點擊 F. 選擇器的權重： 1. 內聯樣式 1,0,0,0 2. id選擇器 0,1,0,0 3. (偽)類選擇器 0,0,1,0 4. 元素選擇器 0,0,0,1 5. 通配選擇器 0,0,0,0 *{} 6. 繼承是樣式 沒有優先級
