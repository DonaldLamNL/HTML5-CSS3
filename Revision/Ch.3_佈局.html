Ch.3 css佈局 A. 盒子模型 - 盒子模型由：內容區+內邊框+邊框+外邊距組成 content + padding + border + margin 1. 內容 content - 以 width 和 height 設置內容區的大小 2. 邊框 border 邊框設計： - border-width 邊框寬度 - border-color 邊框顏色 - border-style 邊框樣式 - 結構：border: width color style; border: 10px yellow solid; 3. 內邊距 padding - 設置邊框離內容區的距離 4. 外邊距 - 設置盒子模型的位置，並不會影響盒子的樣式 B. 盒子模型的佈局 1. 水平佈局 元素在其父元素中水平方向的位置由以下屬性決定： margin-left border-left padding left width padding-right border-right margin-right
